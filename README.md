Задача 3

```
Разработайте программу, в которую пользователь вводит два числа, а программа выводит их произведение. 
При этом программа будет запрашивать ввод чисел у пользователя до тех пор, пока оба введенных числа не окажутся в диапазоне от 0 до 10. 
Если введенные числа больше 10 или меньше 0, программа сообщит пользователю, что введенные числа недопустимы, и попросит ввести два числа снова. 
Если введенные числа находятся в диапазоне от 0 до 10, программа выведет результат их умножения.
Для организации ввода чисел используется бесконечный цикл while и оператор break.
```
