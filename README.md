Задача 4

```
Предположим, что в банке процентная ставка на вклад зависит от его суммы. 
Напишите консольную программу, где пользователь вводит сумму вклада. Если сумма вклада меньше 100, программа начисляет 5%. 
Если сумма находится в диапазоне от 100 до 200, начисляется 7%. При сумме вклада свыше 200 применяется ставка в 10%. В конце программа выведет сумму вклада с начисленными процентами.
Для получения вводимого числа с клавиатуры используйте выражение Convert.ToDouble(Console.ReadLine()).
```
