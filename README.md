Задача 5

```
Напишите консольную программу, которая выводит пользователю сообщение "Введите номер операции: 1. Сложение 2. Вычитание 3. Умножение". 
Пользователь должен ввести номер операции, и в зависимости от выбранного номера программа выведет название соответствующей операции.
Для определения операции по введенному номеру используйте конструкцию switch...case.
Если введенное число не соответствует ни одной операции (например, число 120), программа выведет сообщение о том, что операция неопределена.
```
